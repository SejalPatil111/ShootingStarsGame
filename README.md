# 🌠 Shooting Stars

A simple 2D shooting game built with React Native and Expo — rotate your gun, aim at stars moving across the screen, and fire bullets to score points before you run out of targets.

## How to Run This Project

1. **Clone the repository**
```bash
   git clone https://github.com/YOUR-USERNAME/ShootingStarsGame.git
   cd ShootingStarsGame
```

2. **Install dependencies**
```bash
   npm install
```

3. **Start the development server**
```bash
   npx expo start
```

4. **Run on your phone**
   - Install the **Expo Go** app from the Play Store (Android) or App Store (iOS).
   - Make sure your phone and computer are on the same Wi-Fi network.
   - Scan the QR code shown in the terminal using the Expo Go app (Android) or your Camera app (iOS).

## Gameplay

- Rotate the gun left/right using the ◀ / ▶ buttons
- Tap 🔥 FIRE to shoot at moving stars
- Score increases when you hit a star; Miss increases when a star escapes off-screen
- Clear all 15 stars to see your final score and high score
